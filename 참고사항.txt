- 탑 버튼 구현 완료(고정기능 구현 완료), 다만 디자인 형태가 좀 바뀌어서 이 부분에 대해서 기디분들께 추후 말씀드릴예정
- 네비게이션 바 구현 완료(고정기능 구현 완료) : 로그인 이후 적용되는 부분에 대해서는 추후 JS 적용 후 구현할 예정
- 전체 적인 개발 과정예상은 다음과 같습니다.
1.HTML CSS를 이용한 주요 페이지 기본 레이아웃 구현
2.주요 페이지들에 대한 HTML CSS구현 이후 그를 바탕으로 JS로 동적활동 구현(로그인관련 기능 등)
3.최종적으로 리액트에 모든 내용을 옮겨 라우팅설정(경로설정) 및 멋있는 백엔드 여러분들의 데이터베이스와 연동 후 안정화?

이후 예정 진행 사항
1단계인 주요 페이지에 대한 (HTML&CSS) 적용을 진행합니다.
정이님 - 질문게시판 구현
효원님 - 유저피드 구현

*
html파일은 크게 3가지 구조로 구성되어 있습니다.
header : 네비게이션 바
section(혹은 figure라고 지칭) : 메인 내용들이 들어가는 부분
footer : 화면 하단부분에 들어갈 내용 

footer부분은 현재로서는 쓰이는 곳이 없어 별다른 내용을 따로 추가하지 않았음

그리고 css 파일 중
reset.css는 혹여나 페이지에 남아있을 수 있는 기본 css내용을 전부 초기화시키는 파일로, 그냥 맘 편히 적용해주시면 됩니당!

*
현재 프론트 프로젝트 폴더는 크게 4가지로 구성되어 있습니다.
html / css / js(사용자와의 동적 상호작용을 위한 코드, 추후 개발) / image(UI구현에 필요한 어느정도의 자료)
